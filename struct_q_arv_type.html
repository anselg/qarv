Detailed Description

These types are used by the QArvCamera model and delegate to edit feature node values.

Sometimes, a feature has several possible types (e.g. an enumeration can be either an enumeration, a string or an integer; an integer can be cast to a float etc.), but the delegate needs to be able to identify the type exactly. Therefore, each type is given a distinct class. When deciding what type to return, the model tries to match the highest-level type. Each type also provides its own editing widget.
